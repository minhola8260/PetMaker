# f07-지도 기반 위치 표시 (예정)

- **관련 요구사항**: `FR-07`
- **관련 파일**: 미배포 (차기 개발 예정 사양)

---

## 1. 동작 설명
이 기능은 Google Maps API를 연동하여 지도 화면에서 사용자의 실시간 GPS 위치를 추적 표시하고, 지도 내 펫 출현 중심 반경 영역(서클 오버레이)을 도시하여 반경 진입 시 펫 발견 트리거 및 푸시 알람을 수행하도록 설계되었습니다.

---

## 2. 기술 구현 방식
- **현재 구현 수준**:
  현재는 GPS 위경도 파싱 및 Geocoder를 이용한 구/동 단위 한글 주소 표기가 메인 화면 기상 상세창과 도감 모달 상세창에 구축되어 활용되고 있습니다.
- **향후 기능 확장**:
  - Google Maps SDK for Android를 탑재하여 MapView 기반 지도 컴포넌트를 렌더링할 예정입니다.
  - 사용자 좌표 및 출현지 좌표 사이의 거리 연산(하버사인 공식 등)을 적용해 반경 N미터 진입 감지 백그라운드 서비스를 트리거하도록 고도화 작업을 계획 중입니다.
